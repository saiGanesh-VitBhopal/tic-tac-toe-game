# Tic Tac Toe - Cloud Game

A simple and fun Tic Tac Toe game built with HTML, CSS, and JavaScript. Play against a friend with real-time turn display and game status updates.

## Features

- **Two-Player Gameplay**: Player 1 (X) vs Player 2 (O)
- **Real-Time Turn Display**: Shows whose turn it is
- **Last Move Tracking**: Displays the last move made by each player
- **Win Detection**: Automatically detects winning combinations
- **Draw Detection**: Recognizes when the game ends in a draw
- **Reset Button**: Start a new game anytime
- **Dark Theme**: Modern dark-themed UI inspired by GitHub's design

## How to Play

1. Click on any empty cell to place your mark
2. Player 1 (X) goes first, then Player 2 (O)
3. First player to get three marks in a row (horizontally, vertically, or diagonally) wins!
4. If all cells are filled and no winner is found, it's a draw
5. Click the "Reset Game" button to start a new game

## Winning Combinations

The game checks for winning combinations in:
- Rows: [0,1,2], [3,4,5], [6,7,8]
- Columns: [0,3,6], [1,4,7], [2,5,8]
- Diagonals: [0,4,8], [2,4,6]
